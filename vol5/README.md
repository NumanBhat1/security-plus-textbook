## High-yield formulas & mappings to memorize

- **SLE = AV × EF** (single loss expectancy)
- **ALE = SLE × ARO** (annualized loss expectancy) — buy a control if its cost < the ALE it reduces
- **RTO** = max downtime · **RPO** = max data loss (drives backup frequency) · **MTTR** = repair time · **MTBF** = uptime between failures
- **Risk strategies:** Mitigate (controls) · Transfer (insurance) · Avoid (don't do it) · Accept (exemption/exception)
- **Pen-test environments:** Known = white-box · Partially known = gray-box · Unknown = black-box
- **Recon:** Passive (no contact / OSINT) vs. Active (scanning / detectable)
- **Privacy:** Controller decides purpose/means · Processor acts on its behalf · Data subject = the person · Right to be forgotten = erasure (GDPR)
