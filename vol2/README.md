## High-yield distinctions to memorize

- **Actors:** nation-state (espionage/war, APT) · organized crime (money) · hacktivist (cause) · insider (internal) · unskilled (others' tools) · shadow IT (internal, unsanctioned)
- **Phishing family:** phishing = email · vishing = voice · smishing = SMS
- **SQLi vs. XSS:** SQLi targets the database (server-side) · XSS runs in other users' browsers (client-side)
- **Virus vs. worm:** virus needs a user to run it · worm self-spreads
- **Password spraying vs. brute force:** few passwords × many accounts (evades lockout) vs. many passwords × one account
- **Indicators:** impossible travel = stolen creds · missing logs = anti-forensics · resource inaccessibility = ransomware/DoS
- **Segmentation vs. isolation:** divide into zones vs. fully separate · **allow list** = default-deny
